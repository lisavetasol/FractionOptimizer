FractionOptimizer - calculate optimal LC conditions for fractionation and fraction analysis
===========================================================================================

Requirements
------------
- Python 2.7
- matplotlib
- scipy
- pyteomics
- pyteomics.biolccc


How to use
----------
    
``python FractionOptimizer.py parameters.cfg peptide_list.csv output_dir``
